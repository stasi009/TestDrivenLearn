Boolean Rules

  • true: boolean true, nonempty string (including a string containing only whitespaces), nonzero number, any object
  • false: boolean false, empty string, 0, NaN, null, undefined

Number Rules

  • boolean: true returns 1, false returns 0
  • numeric: return itself
  • undefined: return NaN
  • null: return 0
  • empty string: return 0
  • string consists only number: return number
  • string contains any text: return NaN

parseInt Rules

  • string is null: NaN
  • first non-whitespace is not number: NaN
  • parse until first non-numeric character: return already parsed
  • started with 0 will return different value in different explorer.

parseFloat Rules

  • it can recognize the scientific notation
  • ignore the starting 0 or 0x
